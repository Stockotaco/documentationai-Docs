## Replace snippets: PUT /clients/:id/snippets

Replace all custom code snippets for a client. Snippets are arbitrary HTML or JavaScript injected via remote configuration. After updating, the API rebuilds the client's remote configuration in KV.

### Path parameters

<ParamField path="id" param-type="string" required="true" deprecated="false">
  Client identifier.
</ParamField>

### Body fields

<ParamField body="snippets" param-type="array of objects" required="true" deprecated="false">
  Array of snippet objects. Replaces all existing snippets.
</ParamField>

<ParamField body="snippets[].code" param-type="string" required="true" deprecated="false">
  Raw HTML or JavaScript code to inject.
</ParamField>

<ParamField body="snippets[].description" param-type="string" required="false" deprecated="false">
  Human-readable description. Defaults to `null`.
</ParamField>

<ParamField body="snippets[].enabled" param-type="boolean" required="false" deprecated="false">
  Whether the snippet is active. Defaults to `true`.
</ParamField>

<ParamField body="snippets[].trigger_rules" param-type="object" required="false" deprecated="false">
  Conditional rules controlling when the snippet loads. Defaults to `null` (always loads).
</ParamField>

### Example

<Request show-lines="true">
  ```bash
  curl -X PUT "https://api.mythic-analytics.com/api/v1/clients/acme-retail/snippets" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "snippets": [
        {
          "code": "<script src=\"https://widgets.mythic-analytics.com/checkout.js\"></script>",
          "description": "Checkout widget",
          "enabled": true
        },
        {
          "code": "<div data-mythic-badge=\"acme-retail\"></div>",
          "description": "Profile badge",
          "enabled": true,
          "trigger_rules": { "url_contains": "/profile" }
        }
      ]
    }'
  ```
</Request>
