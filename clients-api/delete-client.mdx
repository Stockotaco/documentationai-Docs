---
title: "Delete Client"
description: >-
  Soft-delete or permanently remove a Mythic Analytics client tenant
  and all associated data.
---

## Delete client: DELETE /clients/:id

Soft-delete by default (sets `status` to `false`). Pass `?hard=true` to permanently remove the client from the database and KV store.

### Path parameters

<ParamField path="id" param-type="string" required="true" deprecated="false">
  Client identifier.
</ParamField>

### Query parameters

<ParamField query="hard" param-type="string" required="false" deprecated="false">
  Set to `true` for permanent deletion. Default is soft delete.
</ParamField>

<Callout kind="danger">
  Hard delete permanently removes the client and all associated data from the database and KV store. This cannot be undone.
</Callout>

### Example (soft delete)

<Request show-lines="true">
  ```bash
  curl -X DELETE "https://api.mythic-analytics.com/api/v1/clients/acme-retail" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY"
  ```
</Request>

<Response show-lines="true" tabs="200">
  ```json
  {
    "success": true,
    "deleted": true,
    "hard": false
  }
  ```
</Response>

### Example (hard delete)

<Request show-lines="true">
  ```bash
  curl -X DELETE "https://api.mythic-analytics.com/api/v1/clients/acme-retail?hard=true" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY"
  ```
</Request>

<Response show-lines="true" tabs="200">
  ```json
  {
    "success": true,
    "deleted": true,
    "hard": true
  }
  ```
</Response>
