## Loader snippets: GET /clients/:id/loader

Retrieve the loader configuration including the publishable key, embed URL, and ready-to-paste HTML snippets.

### Path parameters

<ParamField path="id" param-type="string" required="true" deprecated="false">
  Client identifier.
</ParamField>

### Example

<Request show-lines="true">
  ```bash
  curl "https://api.mythic-analytics.com/api/v1/clients/acme-retail/loader" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY"
  ```
</Request>

<Response show-lines="true" tabs="200">
  ```json
  {
    "success": true,
    "data": {
      "publishable_key": "pk_a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4",
      "embed_url": "https://api.mythic-analytics.com/embed/loader/pk_a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4",
      "iframe_snippet": "<iframe src=\"https://api.mythic-analytics.com/embed/loader/pk_a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4\" style=\"border:0;width:100%;height:600px\" loading=\"lazy\"></iframe>",
      "script_snippet": "..."
    }
  }
  ```
</Response>

<Callout kind="info">
  The `script_snippet` contains a minified loader that creates the SDK global, initializes it with the correct `api_host`, and loads the SDK from the CDN. Use this for the simplest integration path.
</Callout>
