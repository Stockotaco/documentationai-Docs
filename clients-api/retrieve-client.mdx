## Retrieve client: GET /clients/:id

Fetch a single client with its related tags, snippets, and transformers.

### Path parameters

<ParamField path="id" param-type="string" required="true" deprecated="false">
  Client identifier.
</ParamField>

### Example

<Request show-lines="true">
  ```bash
  curl "https://api.mythic-analytics.com/api/v1/clients/acme-retail" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY"
  ```
</Request>
