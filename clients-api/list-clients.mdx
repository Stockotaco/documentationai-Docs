## List clients: GET /clients

Retrieve a paginated list of clients with optional filtering.

### Query parameters

<ParamField query="status" param-type="string" required="false" deprecated="false">
  Filter by active status. Pass `true` or `false`.
</ParamField>

<ParamField query="company_id" param-type="string" required="false" deprecated="false">
  Filter by company identifier.
</ParamField>

<ParamField query="search" param-type="string" required="false" deprecated="false">
  Case-insensitive substring match on client name.
</ParamField>

<ParamField query="limit" param-type="integer" required="false" deprecated="false">
  Maximum results per page. Defaults to `50`, maximum `500`.
</ParamField>

<ParamField query="offset" param-type="integer" required="false" deprecated="false">
  Number of results to skip. Defaults to `0`.
</ParamField>

### Example

<Request show-lines="true">
  ```bash
  curl "https://api.mythic-analytics.com/api/v1/clients?status=true&limit=2&offset=0" \
    -H "Authorization: Bearer YOUR_ADMIN_KEY"
  ```
</Request>
