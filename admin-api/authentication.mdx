## Admin key

Every Admin API endpoint requires an admin API key. Pass it as a bearer token or query parameter — either satisfies the check.

<ParamField header="Authorization" param-type="string" required="true" deprecated="false">
  Bearer token. Format: `Bearer YOUR_ADMIN_KEY`.
</ParamField>

<ParamField query="admin_key" param-type="string" required="false" deprecated="false">
  Alternative to the header for server-to-server or tooling scenarios. Do not expose in client-side code.
</ParamField>

<Callout kind="alert">
  Never expose the admin key in client-side JavaScript, public repositories, or logs. Admin key access should be restricted to trusted backend services and operations staff only.
</Callout>
